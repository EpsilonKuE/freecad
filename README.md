A Geargenerator for freecad
---------------------------

* install:
  * git clone
  * link or copy the FCgear/gear into /freecad/Mod (sudo ln -s (path_to_FCGear)/gear (path_to_freecad)/Mod


* create a gear:
  * open freecad
  * go to the gear workbench
  * create new document
  * create a gear (click on gear symbol)
  * change parameters
